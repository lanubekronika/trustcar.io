# TrustCar.io — Static Site

This is a clean, professional, static website for TrustCar.io.

## Structure
- `public/` — All files served from here (Vercel root)
- `styles/main.css` — Single source of truth for styling
- Pages:
  - index.html (Homepage)
  - products.html (Offers & Pricing)
  - how-it-works.html
  - about.html
  - contact.html

## Design Principles
- Blue professional palette
- Clean typography
- Mobile-first responsive
- Honest, trustworthy copy
- Minimal duplication (copy header/footer pattern when adding pages)

## Deployment
Deploy the `public/` folder to Vercel (or any static host). Set the publish directory to `public`.
